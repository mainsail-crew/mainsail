# Mainsail CNC Extension

## Product Purpose
A CNC-focused control interface built on top of Klipper/Moonraker/Mainsail. Provides specialized panels for machine jogging, offsets, MDI commands, spindle control, and CNC status monitoring. Targets makers and engineers who use Klipper for CNC machines.

## Users
- CNC machine operators (primary)
- Makers and hobbyists
- Machine shop technicians

## Tone
Professional, precise, focused. Actions should feel responsive and intentional. No fluff.

## Register
product

## Anti-references
- Overly decorative UI with gradients and blur effects
- Slow, laggy interactions
- Unclear button purpose or state
- Information overload in panels

## Strategic Principles
1. **Precision over aesthetics** - Controls must be unmistakable in purpose
2. **Quick access to common operations** - Jog, home, offsets, status visible at a glance
3. **Feedback matters** - State changes and command execution should be clear
4. **Responsive controls** - No lag; commands execute immediately
